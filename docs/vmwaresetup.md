# Virtual Machine vmware 17 pro Setup

1.  Sign up at [Broadcom](https://support.broadcom.com/).
2.  Download [VMware Workstation Pro](https://www.vmware.com/products/desktop-hypervisor/workstation-and-fusion).
3.  Download [Ubuntu 22.04.5 LTS](https://releases.ubuntu.com/jammy/).
4.  Create a virtual machine with the following specifications: Disk: 30 GB; RAM: 8 GB; CPU: 4 cores.
5.  Once inside, follow the [Ubuntu Linux environment](docs/) setup guide
